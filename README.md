## Description

The bash script used for scan and parse IP-addresses, ports and MAC-addresses for easy paste to excel.

Parsing Include:

- IP-address
- open TCP ports
- opne UDP ports	
- MAC-address
- NetBIOS-name

## Usage and Result

```bash
./get_hosts.sh --target 192.168.0.2
```
